# Media Rights

The open-source license in this repository covers only original code, schemas,
documentation, and synthetic examples created for this repository.

Third-party media, including robot product images, video frames, source videos,
screenshots, logos, product names, and trademarks, remains owned by the
respective rights holders.

Unless explicitly stated otherwise, third-party media is not covered by this
repository's MIT License.

If a media asset is included for evidence demonstration, it must include:

- Source URL
- Rights holder
- Access date
- License or usage basis
- Whether the asset is included in the open-source license: yes/no

Full third-party source videos must not be uploaded to this repository.
