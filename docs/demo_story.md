# Demo Story

The demo starts from robot product evidence rather than a keyword query.

1. A user provides a product image, product page, spec sheet, demo video reference, or patent drawing.
2. The system organizes the product into clickable components.
3. Patent-sensitive modules are decomposed more deeply than common commodity parts.
4. Each component is connected to observable evidence, inferred technical features, and missing evidence.
5. The workflow produces a patent-risk map and a sample FTO evidence report.

The public UCWS repository demonstrates the workflow with synthetic or
public-reference data. It does not include production Cernify services.
