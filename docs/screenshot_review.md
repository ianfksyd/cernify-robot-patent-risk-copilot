# Screenshot Review

Before publishing screenshots, confirm:

- No account names, emails, session IDs, or customer identifiers
- No browser address bar tokens or private query parameters
- No internal API routes or backend paths
- No customer workspaces or customer reports
- No private source URLs or signed media URLs
- No third-party logo used as project branding
- No unreviewed video frames or product images

The first public release may omit screenshots entirely.
