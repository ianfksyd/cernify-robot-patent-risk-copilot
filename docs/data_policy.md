# Data Policy

This repository must use only:

- Synthetic examples
- Public-reference metadata
- Materials explicitly authorized for public release

Do not include:

- Customer files
- Customer reports
- Customer screenshots
- Private patent datasets
- Internal search logs
- Prompt chains
- Model traces
- Production database exports
- Secrets or credentials
- Full third-party source videos
- Third-party media presented as open-source assets

If there is any uncertainty, omit the material from the public repository.
