# Public Boundary

This repository is a public demo and reference workflow for UCWS.

## Included

- Demo documentation
- Public capability schemas
- Synthetic or public-reference sample inputs and outputs
- Sample component graphs
- Sample evidence graphs
- Sample patent-risk maps
- Sample FTO evidence reports
- Media rights and public boundary documentation

## Not Included

- Cernify production platform
- Production patent retrieval systems
- FTO reasoning engine internals
- Internal APIs
- Private databases or connectors
- Customer workspaces
- Customer files or reports
- Authentication, authorization, billing, task queues, or deployment configuration
- API keys, tokens, secrets, credentials, logs, or model traces
- Full third-party source videos
- Third-party media as reusable open-source assets

## Data Policy

All examples must be synthetic, public-reference, or explicitly authorized for
public release. Do not submit customer data, private analysis, proprietary patent
datasets, private screenshots, logs, credentials, or internal prompts.
